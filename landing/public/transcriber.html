Requirements

  • macOS 11.0 or later
  • ~2GB disk space for models
  • Internet connection for initial setup

Using with Scout

  1. Install the transcriber service using the command above
  2. Open Scout ‚Üí Settings ‚Üí Transcription
  3. Select "External Service" mode
  4. Click "Test Connection" to verify setup

Configuration

The service uses these default ports (configurable via environment variables):

ZMQ_PUSH_PORT=5555    # Audio input
ZMQ_PULL_PORT=5556    # Transcripts output  
ZMQ_CONTROL_PORT=5557 # Control plane

Manual Installation

If you prefer to review the installation process:

  1. Download the installer script
  2. Review the script contents
  3. Run: chmod +x transcriber-install.sh && ./transcriber-install.sh

Uninstall

To remove the transcriber service:

curl -sSf https://scout.arach.dev/transcriber-uninstall.sh | bash
